🔹기술 스택
영역	기술
프론트	HTML, CSS, JavaScript (기초 Vanilla JS)
백엔드	Spring Boot (Java), MySQL or H2
배포	GitHub Pages + Render or EC2
버전관리	GitHub, 커밋 단위 명확히
지도	네이버 지도 API
예약	외부 네이버 예약 링크 연결
